# FMCG Sales Data Analysis & Business Dashboard
## Project Overview
This project focuses on performing end-to-end data analysis on Fast-Moving Consumer Goods (FMCG) sales data to generate actionable business insights. The analysis includes data cleaning, exploratory data analysis (EDA), data visualization, and dashboard development using Python and Power BI.
The goal of this project is to identify sales trends, product performance, pricing behavior, and category distribution to support data-driven business decision making.
---
## 🎯 Project Objectives
- Clean and prepare FMCG sales dataset for analysis
- Perform Exploratory Data Analysis (EDA) using Python
- Identify sales patterns and product performance trends
- Analyze sales distribution across product categories
- Develop an interactive Power BI dashboard
- Generate business insights to improve sales strategy

## 📈 Key Business Insights

Based on analysis, several insights were identified:

- Monthly sales show fluctuation patterns indicating seasonal demand trends.
- Certain product categories contribute significantly to overall revenue.
- Some sales channels outperform others in total sales contribution.
- Higher-priced products tend to generate larger total revenue when supported by demand volume.
- A small number of products contribute disproportionately to total sales.

These insights can support:

- Inventory planning  
- Pricing strategy  
- Channel optimization  
- Product prioritization
